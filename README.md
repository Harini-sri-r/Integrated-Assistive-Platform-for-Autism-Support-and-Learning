## 📌 About the Project

This project is an AI-based web application created to support individuals with Autism Spectrum Disorder (ASD). The goal is to make communication and learning more accessible by providing an interactive digital platform.

The system allows users to interact through text and receive responses that are adjusted based on the emotional tone of their input. It is designed with simplicity so that it can be easily used without technical knowledge.

---

## 🎯 Objective

Many individuals with autism face difficulties in expressing emotions and engaging in conversations. This project aims to:

* Improve communication through guided interaction
* Provide basic emotional understanding using AI
* Create a supportive and easy-to-use environment

## 🚀 How to Run Locally

### Clone the repository

```bash id="c3rvem"
git clone https://github.com/Harini-sri-r/Integrated-Assistive-Platform-for-Autism-Support-and-Learning.git
cd Integrated-Assistive-Platform-for-Autism-Support-and-Learning
```

### Install required packages

```bash id="j6l0eo"
pip install -r requirements.txt
```

### Run the application

```bash id="rcxsnv"
python app.py
```

### Open in browser

http://127.0.0.1:5000/
